# Programa de Formação em Elixir | Teste Técnico

## Objetivo

O objetivo deste projeto é a resolução do Desafio Técnico para o Programa de Formação em Elixir. A resolução de problema foi feita utilizando a linguagem JAVA.

## Descrição

**Imagine uma lista de compras. Ela possui:**

* Itens; 
* Quantidade de cada item; 
* Preço por unidade/peso/pacote de cada item; 
* Desenvolva uma função (ou método) que irá receber uma lista de compras (como a detalhada acima) e uma lista de e-mails. Aqui, cada e-mail representa uma pessoa; 

**A função deve:**

* Calcular a soma dos valores, ou seja, multiplicar o preço de cada item por sua quantidade e somar todos os itens; 
* Dividir o valor de forma igual entre a quantidade de e-mails; 
* Retornar um mapa/dicionário onde a chave será o e-mail e o valor será quanto ele deve pagar nessa conta; 

**Informações**

* Os valores dos itens são representados sempre com números inteiros; 
* Os valores estão expressos em centavos: R$ 1, 00 é igual a 100 (centavos); 

## Tecnologia

* JAVA

* Execute o arquivo `Pacotes de Codigo Fonte/desafiotecnicoelisete/DesafioTecnicoElisete.java`. Ele contém a classe principal (main);

* Para dúvidas quanto à construção da lógica, basta seguir os comentários nos arquivos `.java`;
